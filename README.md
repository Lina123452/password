# password
# Генератор случайных паролей (Random Password Generator)

## Автор
Артём Холстов

## Описание
GUI-приложение на Python + Tkinter для генерации безопасных паролей. Поддерживает настройку длины (ползунок 4-50), выбор символов (чекбоксы: цифры, буквы, спецсимволы), отображение истории в таблице, сохранение в JSON.

## Установка и запуск
1. Клонируйте репозиторий: `git clone https://github.com/ваш-username/password-generator.git`
2. Запустите: `python main.py`

## Функции
- Генерация пароля кнопкой.
- Копирование в буфер.
- История (до 50 записей, сохраняется в history.json).
- Проверки: мин/макс длина, хотя бы один тип символов.

## Примеры использования
- Длина 16, все символы: `A1b@C2d#E3f$`
- Только буквы, длина 8: `AbCdEfGh`

## Требования
Python 3.6+, tkinter (встроен).
